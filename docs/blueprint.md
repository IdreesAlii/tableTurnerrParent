# **App Name**: Tableturnerr Landing

## Core Features:

- Header: Displays the company logo and navigation links.
- Hero Section: Showcases the main value proposition with a headline, subheadline, and call-to-action button.
- Testimonials Marquee: Displays rotating testimonials from satisfied clients.
- What We Do: Highlights the key services offered by the agency using a feature section with hover effects.
- About Tableturnerr: Provides a brief overview of the agency's mission and values.
- Design Work: Showcases examples of the agency's design work through a grid of images or cards.
- About Owner.com: Provides information about Owner.com and its partnership benefits.
- What Owner.com Does: Illustrates the features and benefits of Owner.com using a radial orbital timeline.
- Qualification Criteria: Lists the criteria for restaurants to qualify for the agency's services.

## Style Guidelines:

- Primary color: Black and white with minimal use of blue for small details.
- Background color: White or light grey for a clean and modern look.
- Accent color: Subtle blue for highlighting key elements and calls to action.
- Headline font: 'Poppins', sans-serif, for a geometric, fashionable feel (as seen in the Hero section).
- Body font: 'PT Sans', sans-serif, for a humanist modern, readable style. (Paired with 'Poppins')
- Use clean, minimalist icons from Lucide to represent services and features, aligning with the style of the 21st.dev components.
- Employ a consistent container and spacing system to ensure visual balance and responsiveness, consistent with the 21st.dev components.
- Use subtle animations, like fade-ins and transitions, to enhance user engagement without being distracting. Refer to the 21st.dev components for animation style.
